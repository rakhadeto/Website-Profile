# 🧑‍💻 Website Profile — Naufal Rakhadeto

A professional, cinematic dark-themed personal profile website with 3-tab navigation. Built with pure HTML, CSS & Vanilla JS — no frameworks, no dependencies.

## ✨ Features

- 🌙 Dark cinematic aesthetic with violet/fuchsia accent
- 📱 Fully responsive — mobile, tablet & desktop
- 🗂️ 3 tabs: **Home**, **About**, **Contact**
- 📲 Mobile bottom navigation bar
- ⚡ Pure HTML + CSS + Vanilla JS — zero dependencies
- 🎨 Smooth tab transitions & micro-interactions
- 🌊 Mouse parallax on ambient background blobs
- 💡 Dynamic greeting based on time of day
- 📊 Animated skill bars
- ✨ Hero card 3D tilt on hover (desktop)
- 💫 Ripple click effects

## 📁 File Structure

```
Website-Profile/
├── index.html     # All 3 tabs HTML structure
├── style.css      # Full styling + responsive layout
├── script.js      # Tab switching, animations, interactions
└── README.md
```

## 🗂️ Tab Overview

| Tab | Content |
|-----|---------|
| **Home** | Welcome greeting (time-based), hero card with stats, CTA buttons |
| **About** | Bio, education, skills with bars, experience timeline, projects, fun facts |
| **Contact** | WhatsApp, Email, GitHub, Instagram link cards |

## 🚀 Setup

1. Clone the repo:
   ```bash
   git clone https://github.com/rakhadeto/Website-Profile.git
   cd Website-Profile
   ```

2. Open `index.html` directly in your browser — no build step needed.

## 🌐 Deploy to GitHub Pages

1. Go to **Settings → Pages**
2. Source: `main` branch, `/ (root)`
3. Save → live at `https://rakhadeto.github.io/Website-Profile`

## 🎨 Customization Guide

| What to change | Where |
|----------------|-------|
| Name & tagline | `index.html` → `.hl-name`, `.home-sub` |
| Bio text | `index.html` → `.bio-long` paragraphs |
| WhatsApp number | `index.html` → `href="https://wa.me/62..."` |
| Email | `index.html` → `href="mailto:..."` |
| Instagram handle | `index.html` → `href` & `.link-handle` |
| Skills & percentages | `index.html` → `data-pct` attributes on `.sbi-fill` |
| Accent color | `style.css` → `--accent` variable |
| Add photo | Replace `.hc-img` initials with `<img src="photo.jpg">` |
